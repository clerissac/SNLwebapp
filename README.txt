SNL iCare Online Application v1.8 - Replacement Notice workflow

Changes:
- Replacement Question 2 = Yes triggers a red italic notice.
- Agent must acknowledge that the customer was informed before the Replacement Notice appears.
- Original Replacement Notice page 1 is shown in full.
- Customer chooses whether page 2 should be read aloud.
- If customer declines the reading, applicant initials are required in Signature Handling.
- If customer does not decline, original page 2 is shown for reading aloud.
- Applicant and producer replacement signatures are added to Signature Handling.
- Printed applicant/producer names and signature dates are collected and placed on the replacement PDF.
- Both original Replacement Notice pages are included in the generated packet whenever replacement is triggered.
- Existing approved sections and mappings were not redesigned.
