SNL iCare Online Application v1.9 - Replacement wording and Application Date

Changes:
- Added Application Date to Agent Information.
- Application Date uses mm/dd/yyyy and is entered once.
- The date is reused for the iCare application, Prescription Authorization, Conditional Receipt, and Replacement Notice signature dates.
- Application Date applies to both direct-signature and DocuSign/email-signature packets.
- Removed the visible scanned Replacement Notice page images from the interview.
- Added all original wording from Replacement Notice pages 1 and 2 in a readable web format.
- Fill-in questions and signature fields remain separate from the notice wording.
- Page 2 wording appears only when the customer does not decline having it read aloud.
- The official two-page Replacement Notice PDF is still included in the final packet whenever replacement is triggered.
- Previously approved sections and mappings were left unchanged.
